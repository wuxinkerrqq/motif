# music_narrative_skill.md
> 适用 Agent：Edit Planner Agent
> 作用：理解音乐的叙事功能，指导如何让音乐驱动视觉叙事，而不是让画面被动配音乐

---

## 核心原则

**音乐不是背景，是第二位叙事者。**

音乐和画面的关系有三种模式，Edit Planner 必须根据 `editing_style` 选择正确的模式：

| 模式 | 名称 | 含义 | 适用风格 |
|------|------|------|----------|
| 平行（Parallel） | 音画同向 | 音乐情绪和画面情绪完全一致 | 视觉向、燃向 |
| 对位（Contrapuntal） | 音画反向 | 音乐情绪和画面情绪刻意相反，制造张力或讽刺 | 叙事向、情绪向 |
| 中性（Neutral） | 音画脱离 | 音乐作为纯节奏载体，不承担情绪叙事功能 | 纯卡点向 |

**重要提示：** 不能所有段落都用平行模式，适当的对位会让整体更有层次感和冲击力。

## 一、音乐的三个叙事层次

### 层次一：铺垫情绪（基础用法）

音乐在这个层次上的作用是**建立情感基调**，观众在听到音乐的前几秒就已经知道"接下来要感受什么"。

**使用规则：**
- 每个音乐段落的 `mood` 标签直接对应素材的 `emotion_mood`
- 这是最基础的匹配方式，但也最容易显得平淡
- 单独使用此层次的作品，通常被评价为"合格但缺乏记忆点"

**适用场景：** `editing_style = "visual-driven"`，用户明确要求视觉优先时

---

### 层次二：搭建结构（中级用法）

音乐在这个层次上的作用是**定义叙事骨架**，音乐的段落结构直接决定故事的起承转合节奏。

**核心思想：** 音乐段落 = 叙事章节

```
intro     → 建立世界观，交代背景
verse     → 角色关系建立，日常状态呈现
buildup   → 冲突积累，紧张感上升
chorus    → 情感爆发或关键转折
bridge    → 叙事转折，情绪换挡
drop      → 高潮时刻，最强视觉冲击
outro     → 余韵，主题升华，情感收尾
```

**使用规则：**
- 严格按音乐段落分配 `plot_stage`，不允许跨段落乱序
- `section_boundary`（大段落切换点）必须对应画面的叙事转折
- 同一个音乐段落内部，画面情绪应保持一致性

**适用场景：** `editing_style = "story-driven"`

---

### 层次三：优化节奏（高级用法）

音乐在这个层次上的作用是**精确控制观众的情绪节拍**，每一个 beat、每一次能量变化都对应一个精心设计的视觉动作。

**四种精细化技巧：**

**① Beat Sync（节拍同步）**
每个镜头切换点精确对齐 `beat_array` 中的时间戳。高能量段落对齐强拍（downbeat），低能量段落可以对齐弱拍或完全不卡拍，制造漂浮感。

**② Snap to Beat（吸附算法）**
LLM 选好场景后，由代码强制将切点吸附到最近的 beat 时间戳，容许误差 ±0.2s。不让模型算数学，让代码保证精度。

**③ 能量曲线跟踪（Energy Tracking）**
画面的 `action_density` 变化应跟随音乐的 `energy` 变化曲线：
```
音乐 energy 上升 → 素材 action_density 同步上升
音乐 energy 下降 → 素材 action_density 同步下降
音乐 energy 骤变 → 素材在对应时间戳做切换
```

**④ 特殊事件锚点锁定（Anchor Reservation）**
`special_events` 中标注的特殊时刻（drop/sudden_silence/key_change等）必须优先锁定，其他段落围绕这些锚点填充，而不是反过来。

**适用场景：** 所有风格，叙事和情绪规划完成后的最后一步优化

## 二、专业音乐叙事技法

### 技法一：Leitmotif（主题动机）

**定义：** 与特定人物、情感或叙事概念绑定的反复出现的音乐主题。

**在混剪中的应用：**
- 如果音乐中存在反复出现的旋律片段（通常在 `audio_analyzer` 的段落分析中可以识别），该片段每次出现时应对应相同人物或相同叙事主题的画面
- 例如：每次听到某段旋律，画面都出现 David 和 Lucy 在一起的场景，强化观众的记忆连接
- 这是让观众在情感上"记住"某个角色的最有效手段之一

**检测方法：** 观察 `audio_map.segments` 中是否有 `name` 相同但 `start` 不同的段落，这通常意味着该段落是 Leitmotif 的重现。

---

### 技法二：Mickey Mousing（动作同步）

**定义：** 音乐精确匹配画面中的动作，每一个动作都有对应的音效或节拍呼应。

**使用边界：**
- **适量使用**：高潮打斗场景、强节奏段落中，画面动作精确对齐强拍，效果震撼
- **过度使用**：会显得刻意和幼稚，失去电影感
- **黄金比例**：整片中 Mickey Mousing 的切点占比不超过 30%，其余保持自然节奏

**适用场景：** `drop`、`chorus` 段落，`action_density >= 7` 的素材

---

### 技法三：Contrapuntal（对位法）

**定义：** 音乐情绪与画面情绪刻意相反，制造张力、讽刺或意想不到的情绪深度。

**经典案例：**
- 欢快轻松的音乐 + 悲伤或残酷的画面 → 制造强烈反差和心理冲击
- 平静克制的音乐 + 高强度动作画面 → 制造压抑感和冷峻感

**在混剪中的使用时机：**
- 叙事向混剪的转折点：用对位打破观众的预期，制造"意料之外"的情绪冲击
- 不建议在视觉向混剪中使用，因为观众期望的是音画同步的爽感

**注意：** 对位不是"随机错配"，必须有明确的叙事意图。使用前在 `edit_plan` 中标注原因。

---

### 技法四：音乐心理学的三条原则

基于音乐心理学研究，以下三条原则在混剪中具有普遍适用性：

**① 调性与情绪的对应关系**
```
大调（Major key）→ 喜悦、希望、胜利、温暖
小调（Minor key）→ 悲伤、紧张、神秘、内省
转调（Key change）→ 情绪层级跃升，是最强的情绪提升工具
```
当 `audio_analyzer` 检测到 `key_change` 时，对应画面必须是全片情绪强度最高的时刻之一。

**② 速度与能量感知**
```
BPM > 140  → 紧张、兴奋、紧迫 → 快切，action_density 高
BPM 80-140 → 中性、日常、叙事 → 中等切换频率
BPM < 80   → 沉重、悲伤、宏大 → 长镜头，action_density 低
```

**③ 音量动态与情绪强度**
音乐的 RMS 能量值（`energy_curve`）直接影响观众的生理唤醒水平：
- 能量高（>0.7）→ 观众处于高度唤醒状态，此时快切和强冲击画面效果最好
- 能量低（<0.3）→ 观众处于低唤醒状态，慢节奏和细腻情感画面效果更好
- 能量骤变 →  观众的注意力瞬间被抓住，是切换叙事方向的最佳时机

---

### 技法五：Silence as Storytelling（沉默叙事）

**核心理念：** 沉默不是音乐的缺席，而是音乐叙事最有力的武器之一。

**三种沉默的叙事功能：**

**① 蓄力型沉默（Pre-climax silence）**
高能段落前的短暂静音，让观众的期待感在静默中积累到极限，然后一击即发。
- 对应 `special_events` 中的 `sudden_silence`
- 画面：定格、特写、人物表情
- 时长建议：1-3 秒，不宜过长

**② 消化型沉默（Post-climax silence）**
高潮过后的留白，给观众情绪消化的时间，避免情感疲劳。
- 对应 `outro` 段落的开始阶段
- 画面：慢动作、空镜、渐黑

**③ 强调型沉默（Emphasis silence）**
在重要台词或情绪转折点前短暂去除音乐，让画面或台词的重量被直接感受到（若有台词素材）。

## 三、不同音乐类型的叙事策略

### 日系 OST（泽野弘之、林ゆうき等）
**特征：** 弦乐主导，动态范围极大，频繁出现 sudden_silence + explosion 结构
**叙事策略：**
- 平静段落：长镜头，情感积累，人物关系铺垫
- buildup：镜头开始紧张，人物状态发生变化
- sudden_silence：最强情绪画面的前一秒定格
- 爆发点：全片最震撼的单帧画面，精确对齐

### 电子音乐（EDM、Future Bass、Hybrid Trap）
**特征：** 结构清晰（intro-buildup-drop-outro），drop 前有明显的 riser
**叙事策略：**
- drop 是绝对的视觉焦点，必须先锁定最高能量素材
- buildup 阶段画面切换频率逐步加密，制造视觉加速感
- drop 之后保持高密度快切，不能突然放缓

### 流行歌曲（J-pop、K-pop、欧美流行）
**特征：** 结构规整，通常有多次 chorus，key_change 多出现在最后一次 chorus 前
**叙事策略：**
- 第一次 chorus：情绪初步爆发，但不是最高点
- bridge（桥段）：情绪暂时回落，为最后高潮蓄力
- key_change + final chorus：全片情绪顶点，配合最重要的叙事画面

### 后摇 / 氛围音乐（Sigur Rós、mogwai 风格）
**特征：** 无明显节拍，以音色和音量变化驱动情绪，`beat_array` 参考价值低
**叙事策略：**
- 放弃 `snap_to_beats`，改用 `energy_curve` 作为切换依据
- 镜头切换以情绪共鸣为主，不强调节拍对齐
- 长镜头为主，让音乐的渐变感带动画面

### 交响乐 / 电影配乐（Hans Zimmer、John Williams 风格）
**特征：** Leitmotif 明显，动态范围极大，经常出现 tempo_change 和 mode_shift
**叙事策略：**
- 优先识别 Leitmotif，将其与固定人物或主题绑定
- `mode_shift`（大小调切换）必须对应叙事的情绪转向
- 高潮段落的弦乐爆发对应最宏大的视觉画面（大景别、群像、史诗感）

## 四、音乐叙事的常见错误

| 错误 | 表现 | 后果 | 修正方法 |
|------|------|------|----------|
| 全程平行 | 音乐悲伤画面就悲伤，全程一致 | 缺乏层次，观众情绪麻木 | 在转折点引入对位 |
| 节奏断裂 | 高能量音乐段落用了低 action_density 素材 | 音画脱节，观众困惑 | 严格匹配 energy 和 action_density |
| 提前爆发 | 高潮素材出现在开头或中间段 | 后续无法升级，全片平淡 | 硬性约束 is_climax_material |
| 忽视沉默 | 把 sudden_silence 当普通段落填充 | 错过最强情绪锚点 | 将 special_events 优先级设为最高 |
| 过度 Mickey Mousing | 每个动作都精确对齐节拍 | 显得刻意，失去电影感 | 控制在 30% 以内 |
| 忽视 Leitmotif | 同一旋律主题出现时配了不同人物 | 破坏观众的记忆连接 | 检测并绑定重复出现的音乐段落 |
